Sender Console

Send plaintext through the local sender agent, which encrypts and posts to the relay.

Payload

Status

This page calls the local sender agent at /send, which encrypts with libsignal and posts to the relay. 
Idle.

Usage

  1. Run pnpm sender -- --serve to start the local sender agent.
  2. Serve this file from http://localhost:3001.
  3. Set Server URL to match the relay.
  4. Type plaintext, click Send Message, and check the status panel.
